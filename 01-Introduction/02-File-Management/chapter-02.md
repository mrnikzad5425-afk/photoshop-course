# 📁 فصل ۲: مدیریت فایل‌ها در Adobe Photoshop

## مقدمه

مدیریت صحیح فایل‌ها یکی از مهم‌ترین مهارت‌های یک طراح حرفه‌ای است.

بسیاری از مشکلات طراحان نه به دلیل ضعف در طراحی، بلکه به دلیل نامرتب بودن فایل‌ها، ذخیره‌سازی اشتباه، انتخاب فرمت نامناسب و مدیریت ضعیف پروژه‌ها ایجاد می‌شود.

در این فصل یاد می‌گیریم چگونه از همان ابتدا یک پروژه فتوشاپ را به شکل استاندارد ایجاد، ذخیره و مدیریت کنیم.

---

# اهداف یادگیری

در پایان این فصل شما قادر خواهید بود:

- یک پروژه استاندارد فتوشاپ ایجاد کنید.
- تنظیمات صحیح فایل جدید را انتخاب کنید.
- تفاوت حالت‌های رنگی RGB و CMYK را بدانید.
- فایل‌ها را با فرمت مناسب ذخیره کنید.
- پروژه‌های خود را به شکل حرفه‌ای مدیریت کنید.
- فایل مناسب برای وب و چاپ خروجی بگیرید.

---

# درس ۱: ساخت یک پروژه استاندارد

قبل از شروع هر طراحی، باید یک ساختار مناسب برای پروژه ایجاد کنیم.

یک پروژه حرفه‌ای معمولاً شامل پوشه‌های زیر است:

```
Project-Name

│
├── PSD
│
├── Images
│
├── Fonts
│
├── Export
│
└── Resources
```

توضیح پوشه‌ها:

## PSD

محل ذخیره فایل اصلی فتوشاپ.

مثال:

```
Poster-Design.psd
```

---

## Images

تصاویر استفاده شده در پروژه.

مثال:

```
background.jpg
product.png
```

---

## Fonts

فونت‌های استفاده شده در طراحی.

---

## Export

فایل‌های نهایی خروجی گرفته شده.

مثال:

```
poster-final.jpg
banner.png
```

---

## Resources

فایل‌های کمکی مانند:

- آیکون‌ها
- براش‌ها
- تکسچرها
- فایل‌های دانلودی

---

# درس ۲: ساخت فایل جدید (New Document)

برای ایجاد یک فایل جدید:

از منوی بالا انتخاب کنید:

```
File → New
```

یا از میانبر:

```
Ctrl + N
```

استفاده کنید.

---

پنجره New Document شامل بخش‌های مهمی است:

---

# Width و Height

اندازه تصویر را مشخص می‌کنند.

مثال برای شبکه اجتماعی:

```
Width: 1080 px
Height: 1080 px
```

مناسب برای پست مربعی اینستاگرام.

---

مثال برای پوستر:

```
Width: 2480 px
Height: 3508 px
```

---

# Resolution

رزولوشن فایل را مشخص می‌کند.

تنظیمات معمول:

## طراحی برای نمایشگر:

```
72 PPI
```

## طراحی برای چاپ:

```
300 PPI
```

---

# Color Mode

حالت رنگ پروژه را مشخص می‌کند.

مهم‌ترین حالت‌ها:

- RGB
- CMYK
- Grayscale

---

# درس ۳: تفاوت RGB و CMYK

## RGB

مخفف:

```
Red
Green
Blue
```

این مدل برای نمایشگرها استفاده می‌شود.

کاربرد:

- سایت‌ها
- موبایل
- شبکه‌های اجتماعی
- طراحی دیجیتال

مزایا:

✅ رنگ‌های روشن‌تر  
✅ مناسب صفحه نمایش

---

## CMYK

مخفف:

```
Cyan
Magenta
Yellow
Key (Black)
```

برای چاپ استفاده می‌شود.

کاربرد:

- کارت ویزیت
- بروشور
- مجله
- پوستر چاپی

مزایا:

✅ مناسب چاپ حرفه‌ای

---

# نکته حرفه‌ای

اگر پروژه شما برای اینترنت است:

```
RGB
```

انتخاب کنید.

اگر پروژه برای چاپ است:

```
CMYK
```

انتخاب کنید.

---

# درس ۴: ذخیره کردن فایل‌ها

برای ذخیره پروژه:

```
File → Save
```

میانبر:

```
Ctrl + S
```

---

برای ذخیره با نام جدید:

```
File → Save As
```

میانبر:

```
Ctrl + Shift + S
```

---

# فرمت PSD

فرمت اصلی فتوشاپ:

```
.psd
```

است.

مزایا:

- حفظ لایه‌ها
- حفظ افکت‌ها
- امکان ویرایش دوباره

همیشه فایل اصلی پروژه را PSD نگه دارید.

---

# درس ۵: نام‌گذاری حرفه‌ای فایل‌ها

نام فایل‌ها باید واضح باشد.

نام اشتباه:

```
final.psd
```

یا:

```
design-new-final2.psd
```

---

نام حرفه‌ای:

```
Poster_CoffeeShop_v01.psd
```

یا:

```
Instagram_Post_Product_v02.psd
```

---

روش نسخه‌بندی:

```
v01
v02
v03
```

یعنی نسخه اول، دوم و سوم پروژه.

---

# درس ۶: خروجی گرفتن برای وب

برای ذخیره تصویر مناسب اینترنت:

از مسیر زیر استفاده کنید:

```
File → Export → Export As
```

فرمت‌های مناسب:

## JPG

برای:

- عکس‌ها
- تصاویر با حجم کمتر

---

## PNG

برای:

- لوگو
- تصاویر دارای شفافیت

---

تنظیمات پیشنهادی:

برای شبکه اجتماعی:

```
Format: JPG
Quality: High
```

---

# درس ۷: آماده‌سازی فایل برای چاپ

قبل از چاپ بررسی کنید:

## اندازه فایل

درست باشد.

---

## رزولوشن

حداقل:

```
300 PPI
```

باشد.

---

## حالت رنگ

روی:

```
CMYK
```

باشد.

---

## خروجی چاپ

فرمت‌های رایج:

- TIFF
- PDF
- PSD

---

# درس ۸: مدیریت حرفه‌ای پروژه‌ها

یک طراح حرفه‌ای همیشه:

✅ فایل‌ها را مرتب نگه می‌دارد.

✅ نام لایه‌ها را تغییر می‌دهد.

✅ نسخه‌های مختلف ذخیره می‌کند.

✅ فایل اصلی را حذف نمی‌کند.

✅ از پروژه‌ها نسخه پشتیبان تهیه می‌کند.
