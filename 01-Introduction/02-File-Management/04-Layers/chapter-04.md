# 🧩 فصل ۴: لایه‌ها (Layers) در Adobe Photoshop

## مقدمه

لایه‌ها مهم‌ترین بخش فتوشاپ هستند.

تقریباً تمام پروژه‌های حرفه‌ای در فتوشاپ بر اساس سیستم لایه‌ها ساخته می‌شوند. اگر مفهوم لایه‌ها را به خوبی یاد بگیرید، کنترل بسیار بیشتری روی طراحی، ویرایش و مدیریت پروژه‌های خود خواهید داشت.

در این فصل یاد می‌گیریم چگونه با لایه‌ها کار کنیم، آن‌ها را مدیریت کنیم، مرتب کنیم و از قابلیت‌های حرفه‌ای آن‌ها استفاده کنیم.

---

# اهداف یادگیری

در پایان این فصل شما قادر خواهید بود:

- مفهوم Layer را درک کنید.
- لایه‌های جدید ایجاد و مدیریت کنید.
- ترتیب و ساختار لایه‌ها را کنترل کنید.
- از Group برای سازمان‌دهی پروژه استفاده کنید.
- با Opacity و Blending Mode کار کنید.
- لایه‌ها را قفل، مخفی و کپی کنید.
- پروژه‌های حرفه‌ای با ساختار استاندارد ایجاد کنید.

---

# درس ۱: Layer چیست؟

Layer به معنی لایه است.

در فتوشاپ هر عنصر می‌تواند در یک لایه جدا قرار بگیرد.

مثال:

یک پوستر شامل:

```
Layer 4 : Text
Layer 3 : Logo
Layer 2 : Image
Layer 1 : Background
```

است.

هر لایه مانند یک صفحه شفاف روی لایه‌های دیگر قرار می‌گیرد.

---

# چرا از لایه‌ها استفاده می‌کنیم؟

استفاده از لایه‌ها باعث می‌شود:

✅ هر بخش جداگانه ویرایش شود.

✅ تغییرات بدون خراب کردن تصویر اصلی انجام شود.

✅ پروژه مرتب و قابل مدیریت باشد.

✅ امکان ایجاد افکت‌های حرفه‌ای وجود داشته باشد.

---

# درس ۲: آشنایی با Layers Panel

پنل Layers معمولاً در سمت راست فتوشاپ قرار دارد.

اگر پنل را نمی‌بینید:

مسیر:

```
Window → Layers
```

---

بخش‌های مهم پنل Layers:

## نام لایه

هر لایه باید نام مشخص داشته باشد.

مثال:

```
Background
Product
Shadow
Text
```

---

## Thumbnail

تصویر کوچک نمایشی از محتوای لایه است.

---

## Visibility (علامت چشم)

با کلیک روی چشم می‌توانیم یک لایه را:

- نمایش دهیم.
- مخفی کنیم.

---

## Lock

برای جلوگیری از تغییرات ناخواسته استفاده می‌شود.

---

# درس ۳: ساخت لایه جدید

روش اول:

از پایین پنل Layers روی:

```
New Layer
```

کلیک کنید.

---

روش دوم:

میانبر:

```
Ctrl + Shift + N
```

---

نام‌گذاری لایه:

مثال:

```
Shadow
```

بهتر از:

```
Layer 1
```

است.

---

# درس ۴: حذف، کپی و جابه‌جایی لایه‌ها

## حذف Layer

لایه را انتخاب کنید.

سپس:

```
Delete Layer
```

را بزنید.

---

## کپی Layer

میانبر:

```
Ctrl + J
```

است.

کاربرد:

- ساخت نسخه پشتیبان
- ایجاد افکت‌ها

---

## جابه‌جایی Layer

با Drag کردن در پنل Layers می‌توان ترتیب لایه‌ها را تغییر داد.

مثال:

قبل:

```
Text
Image
Background
```

بعد:

```
Image
Text
Background
```

---

# درس ۵: ترتیب لایه‌ها

ترتیب لایه‌ها اهمیت زیادی دارد.

لایه بالاتر:

روی لایه‌های پایین‌تر نمایش داده می‌شود.

مثال:

```
Text
Image
Background
```

متن روی تصویر قرار دارد.

---

# درس ۶: Group کردن لایه‌ها

وقتی پروژه بزرگ می‌شود، تعداد لایه‌ها زیاد می‌شود.

برای مرتب‌سازی از Group استفاده می‌کنیم.

میانبر:

```
Ctrl + G
```

مثال:

```
Poster

├── Text Group
│   ├── Title
│   └── Subtitle
│
├── Image Group
│   ├── Product
│   └── Shadow
│
└── Background
```

---

# درس ۷: Opacity

Opacity میزان شفافیت لایه را مشخص می‌کند.

مقدار:

```
100%
```

یعنی کاملاً واضح.

---

مثال:

```
Opacity: 50%
```

یعنی لایه نیمه شفاف است.

کاربرد:

- ایجاد سایه
- افکت نور
- ترکیب تصاویر

---

# درس ۸: Fill

Fill شبیه Opacity است اما فقط محتوای لایه را تغییر می‌دهد.

تفاوت:

Opacity:

تمام لایه و افکت‌ها را تغییر می‌دهد.

Fill:

فقط محتوای اصلی لایه را تغییر می‌دهد.

---

# درس ۹: Blending Mode

Blending Mode نحوه ترکیب شدن لایه‌ها با یکدیگر را مشخص می‌کند.

محل قرارگیری:

بالای پنل Layers.

---

حالت‌های مهم:

## Normal

حالت معمولی.

---

## Multiply

برای:

- سایه‌ها
- تیره کردن تصاویر

---

## Screen

برای:

- نور
- روشن‌تر کردن تصاویر

---

## Overlay

برای:

- افزایش کنتراست
- افکت‌های حرفه‌ای

---

## Soft Light

برای ترکیب نرم تصاویر.

---

# درس ۱۰: انواع Layer

فتوشاپ انواع مختلفی از لایه‌ها دارد.

---

## Background Layer

لایه پس‌زمینه.

---

## Text Layer

لایه مخصوص متن.

---

## Shape Layer

برای اشکال برداری.

---

## Adjustment Layer

برای تنظیم نور و رنگ بدون خراب کردن تصویر اصلی.

---

## Smart Object Layer

برای ویرایش غیرمخرب.

(در فصل Smart Objects به صورت کامل بررسی می‌شود.)

---

# درس ۱۱: لایه‌های غیرمخرب (Non-Destructive Editing)

یکی از اصول طراحی حرفه‌ای:

```
ویرایش بدون تخریب تصویر اصلی
```

است.

روش‌های آن:

- استفاده از Adjustment Layer
- استفاده از Smart Object
- نگه داشتن نسخه اصلی

---

# تمرین‌های فصل

## تمرین ۱

یک فایل جدید بسازید.

چهار لایه ایجاد کنید:

```
Background
Shape
Image
Text
```

---

## تمرین ۲

روی یک تصویر:

- Opacity را تغییر دهید.
- Blending Mode های مختلف را امتحان کنید.

---

## تمرین ۳

یک پروژه کوچک بسازید و لایه‌ها را گروه‌بندی کنید.

ساختار:

```
Text Group
Image Group
Background
```

---

# پروژه عملی فصل

## طراحی یک پوستر ساده با لایه‌ها

ایجاد کنید:

- یک پس‌زمینه
- یک تصویر اصلی
- یک عنوان
- یک زیرعنوان
- یک افکت سایه

ساختار نهایی:

```
Poster Project

├── Text Group
│   ├── Title
│   └── Subtitle
│
├── Image Group
│   ├── Main Image
│   └── Shadow
│
└── Background
```

فایل را ذخیره کنید:

```
Layer_Project_01.psd
```

---

# نکات حرفه‌ای

⭐ همیشه نام لایه‌ها را تغییر دهید.

⭐ پروژه‌های حرفه‌ای بدون مدیریت لایه‌ها امکان‌پذیر نیستند.

⭐ از Group برای پروژه‌های بزرگ استفاده کنید.

⭐ فایل PSD را با تمام لایه‌ها نگه دارید.

⭐ قبل از تغییرات بزرگ از لایه‌ها نسخه کپی بگیرید.
