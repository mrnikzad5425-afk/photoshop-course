# ✂️ فصل ۳: ابزارهای انتخاب (Selection Tools) در Adobe Photoshop

## مقدمه

انتخاب کردن (Selection) یکی از مهم‌ترین مهارت‌ها در فتوشاپ است.

تقریباً در تمام پروژه‌های حرفه‌ای، قبل از ویرایش یک بخش از تصویر، تغییر رنگ، حذف پس‌زمینه، جابه‌جایی عناصر یا ایجاد افکت، ابتدا باید قسمت مورد نظر را انتخاب کنیم.

در این فصل با ابزارهای مختلف انتخاب در فتوشاپ آشنا می‌شویم و یاد می‌گیریم چگونه انتخاب‌های دقیق و حرفه‌ای ایجاد کنیم.

---

# اهداف یادگیری

در پایان این فصل شما قادر خواهید بود:

- مفهوم Selection را در فتوشاپ درک کنید.
- با ابزارهای مختلف انتخاب کار کنید.
- انتخاب‌های ساده و پیچیده ایجاد کنید.
- انتخاب‌ها را اصلاح و بهینه کنید.
- از قابلیت‌های حرفه‌ای انتخاب برای پروژه‌های واقعی استفاده کنید.

---

# درس ۱: مفهوم Selection چیست؟

Selection یعنی مشخص کردن یک بخش از تصویر برای انجام تغییرات.

وقتی بخشی از تصویر را انتخاب می‌کنیم، فتوشاپ فقط روی همان قسمت تغییرات را اعمال می‌کند.

مثال:

فرض کنید یک عکس محصول دارید و می‌خواهید فقط رنگ محصول را تغییر دهید.

ابتدا:

```
انتخاب محصول
↓
تغییر رنگ
↓
حفظ بقیه تصویر
```

---

# روش‌های نمایش انتخاب

فتوشاپ قسمت انتخاب شده را با خط متحرکی نمایش می‌دهد که به آن:

```
Marching Ants
```

گفته می‌شود.

M

# Rectangular Marquee Tool

ایجاد انتخاب مستطیلی.

کاربرد:

- برش بخش‌های مربع یا مستطیل
- طراحی کارت‌ها
- انتخاب قسمت‌های صاف تصویر

---

# Elliptical Marquee Tool

ایجاد انتخاب دایره‌ای یا بیضی.

کاربرد:

- انتخاب چشم
- ساخت دایره
- ایجاد ماسک‌های ساده

---

## Add Selection

## Subtract Selection

## Intersect Selection

# درس ۳: Lasso Tool

میانبر:

```
L
```

ابزار Lasso برای انتخاب آزاد استفاده می‌شود.

---

# انواع Lasso

## Lasso Tool

انتخاب دستی با کشیدن موس.

مناسب برای:

- انتخاب‌های سریع
- کارهای ساده

---

## Polygonal Lasso Tool

انتخاب با خطوط مستقیم.

مناسب برای:

- اشکال هندسی
- اجسام با لبه‌های صاف

---

## Magnetic Lasso Tool

به صورت هوشمند لبه‌های تصویر را تشخیص می‌دهد.

مناسب برای:

- جدا کردن سوژه‌ها
- انتخاب اشیای مشخص

---

# درس ۴: Quick Selection Tool

میانبر:

```
W
```

یکی از پرکاربردترین ابزارهای انتخاب است.

این ابزار با تشخیص رنگ و بافت، قسمت‌های مشابه را انتخاب می‌کند.

کاربرد:

- حذف پس‌زمینه
- انتخاب افراد
- جدا کردن محصولات

---

# تنظیمات Quick Selection

## Size

اندازه قلم انتخاب.

---

## Add To Selection

اضافه کردن قسمت‌های جدید.

---

## Subtract From Selection

حذف بخش‌های اضافی.

---

# درس ۵: Magic Wand Tool

میانبر:

```
W
```

(در گروه Quick Selection قرار دارد)

این ابزار بر اساس رنگ انتخاب می‌کند.

مثال:

اگر یک پس‌زمینه سفید داشته باشیم:

```
Magic Wand
↓
کلیک روی سفید
↓
انتخاب تمام بخش‌های مشابه
```

---

# تنظیم مهم Magic Wand

## Tolerance

میزان حساسیت انتخاب رنگ.

عدد پایین:

```
10-20
```

انتخاب دقیق‌تر.

عدد بالا:

```
50-100
```

انتخاب گسترده‌تر.

---

# درس ۶: Object Selection Tool

این ابزار برای انتخاب سریع اشیا طراحی شده است.

فتوشاپ با کمک هوش مصنوعی می‌تواند سوژه‌ها را تشخیص دهد.

کاربرد:

- انتخاب انسان
- انتخاب محصول
- انتخاب حیوانات
- حذف پس‌زمینه

---

# درس ۷: Select and Mask

یکی از مهم‌ترین بخش‌های انتخاب حرفه‌ای.

مسیر:

```
Select → Select and Mask
```

با این بخش می‌توانیم انتخاب‌ها را دقیق‌تر کنیم.

---

کاربردهای مهم:

- جدا کردن مو
- حذف پس‌زمینه پیچیده
- اصلاح لبه‌ها

---

# ابزارهای داخل Select and Mask

## Radius

تشخیص بهتر لبه‌ها.

---

## Smooth

صاف کردن لبه‌های انتخاب.

---

## Feather

نرم کردن لبه‌ها.

---

## Contrast

افزایش وضوح لبه‌ها.

---

## Shift Edge

جلو یا عقب بردن محدوده انتخاب.

---

# درس ۸: Feather و اصلاح انتخاب

گاهی انتخاب‌ها دارای لبه‌های خیلی تیز هستند.

برای طبیعی‌تر شدن می‌توانیم از:

```
Select → Modify → Feather
```

استفاده کنیم.

مثال:

```
Feather: 5 px
```

باعث نرم شدن لبه‌ها می‌شود.

---

# درس ۹: ذخیره انتخاب‌ها

اگر یک انتخاب مهم ایجاد کرده‌ایم، می‌توانیم آن را ذخیره کنیم.

مسیر:

```
Select → Save Selection
```

مزیت:

بعداً بدون انتخاب دوباره می‌توانیم آن را استفاده کنیم.

---

# تمرین‌های فصل

## تمرین ۱

یک تصویر پرتره انتخاب کنید.

با ابزارهای زیر سوژه را انتخاب کنید:

- Quick Selection Tool
- Object Selection Tool
- Select and Mask

---

## تمرین ۲

یک تصویر محصول انتخاب کنید.

پس‌زمینه را جدا کنید.

خروجی:

```
PNG با پس‌زمینه شفاف
```

---

## تمرین ۳

با Marquee Tool:

- یک مربع
- یک دایره

ایجاد کنید و رنگ آن‌ها را تغییر دهید.

---

# پروژه عملی فصل

## حذف پس‌زمینه یک محصول

مراحل:

1. باز کردن تصویر محصول.
2. انتخاب سوژه.
3. اصلاح لبه‌ها با Select and Mask.
4. ایجاد لایه جدید.
5. قرار دادن پس‌زمینه جدید.
6. ذخیره فایل PSD.

ساختار لایه‌ها:

```
Background
Product
Original Image
```

---

# نکات حرفه‌ای

⭐ همیشه قبل از حذف بخش‌های تصویر، انتخاب ایجاد کنید.

⭐ برای انتخاب‌های پیچیده از یک ابزار استفاده نکنید؛ ترکیب ابزارها نتیجه بهتری دارد.

⭐ Select and Mask یکی از مهم‌ترین مهارت‌های فتوشاپ حرفه‌ای است.

⭐ انتخاب دقیق باعث می‌شود نتیجه نهایی طبیعی‌تر باشد.

⭐ میانبرهای انتخاب را یاد بگیرید تا سرعت کار افزایش پیدا کند.
