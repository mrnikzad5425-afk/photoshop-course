# 🎨 فصل ۱: آشنایی با Adobe Photoshop

## مقدمه

به اولین فصل از دوره جامع آموزش Adobe Photoshop خوش آمدید.

در این فصل با مفاهیم پایه فتوشاپ، کاربردهای آن، محیط کار نرم‌افزار، ابزارهای اصلی و فرمت‌های تصویری آشنا می‌شویم.

هدف این فصل این است که قبل از شروع طراحی، شناخت مناسبی از محیط فتوشاپ و اصول اولیه کار با تصاویر دیجیتال پیدا کنید.

---

# اهداف یادگیری

در پایان این فصل شما قادر خواهید بود:

- فتوشاپ و کاربردهای آن را بشناسید.
- بخش‌های مختلف محیط کاری فتوشاپ را توضیح دهید.
- ابزارهای اصلی فتوشاپ را بشناسید.
- با انواع فرمت‌های تصویری آشنا شوید.
- مفهوم Pixel و Resolution را درک کنید.
- یک فایل استاندارد برای شروع پروژه ایجاد کنید.

---

# درس ۱: فتوشاپ چیست؟

Adobe Photoshop یکی از قدرتمندترین نرم‌افزارهای طراحی، ویرایش و پردازش تصاویر دیجیتال در جهان است.

فتوشاپ در زمینه‌های مختلف استفاده می‌شود:

- طراحی گرافیک
- ویرایش و اصلاح عکس
- طراحی پوستر و تبلیغات
- تولید محتوای شبکه‌های اجتماعی
- طراحی رابط کاربری
- ساخت تصاویر ترکیبی (Photo Manipulation)
- آماده‌سازی فایل‌های چاپی

---

# تاریخچه کوتاه فتوشاپ

فتوشاپ در ابتدا توسط توماس نال و جان نال ساخته شد و بعدها توسط شرکت Adobe توسعه پیدا کرد.

امروزه فتوشاپ یکی از مهم‌ترین ابزارهای صنعت طراحی، عکاسی و تبلیغات محسوب می‌شود.

---

# درس ۲: آشنایی با محیط کار فتوشاپ

محیط فتوشاپ از بخش‌های مختلفی تشکیل شده است.

## ۱. Menu Bar (نوار منو)

در بالای صفحه قرار دارد.

بخش‌های مهم:

- File
- Edit
- Image
- Layer
- Select
- Filter
- View
- Window
- Help

مثال:

```
File → Open
```

باز کردن یک فایل جدید.

```
File → Save As
```

ذخیره فایل با نام یا فرمت جدید.

---

# ۲. Toolbar (نوار ابزار)

نوار ابزار در سمت چپ محیط فتوشاپ قرار دارد.

## Move Tool (V)

برای جابه‌جایی لایه‌ها و عناصر استفاده می‌شود.

---

## Marquee Tool (M)

برای ایجاد انتخاب‌های هندسی مانند مربع و دایره استفاده می‌شود.

---

## Lasso Tool (L)

برای انتخاب آزاد بخش‌هایی از تصویر استفاده می‌شود.

---

## Crop Tool (C)

برای برش و تغییر اندازه تصویر استفاده می‌شود.

---

## Brush Tool (B)

برای نقاشی، رنگ‌آمیزی و ایجاد افکت استفاده می‌شود.

---

## Type Tool (T)

برای اضافه کردن متن به پروژه استفاده می‌شود.

---

## Pen Tool (P)

یکی از ابزارهای حرفه‌ای فتوشاپ برای ایجاد مسیرهای دقیق و طراحی اشکال پیچیده است.

---

# ۳. Panels (پنل‌ها)

پنل‌ها در سمت راست محیط فتوشاپ قرار دارند.

## Layers Panel

مهم‌ترین بخش فتوشاپ است.

تمام عناصر پروژه در قالب لایه‌ها مدیریت می‌شوند.

مثال:

```
Layer 3 : Text
Layer 2 : Image
Layer 1 : Background
```

مزایای استفاده از لایه‌ها:

- ویرایش آسان‌تر
- کنترل بیشتر روی پروژه
- حفظ کیفیت طراحی

---

## History Panel

لیست تغییرات انجام شده را نمایش می‌دهد.

با استفاده از این پنل می‌توان به مراحل قبلی بازگشت.

---

## Properties Panel

تنظیمات مربوط به ابزارها و لایه‌های انتخاب شده را نمایش می‌دهد.

---

# درس ۳: Workspace (فضای کاری)

فتوشاپ دارای محیط‌های کاری مختلفی است:

- Essentials
- Photography
- Graphic and Web
- Painting
- Motion

برای شروع پیشنهاد می‌شود از Workspace پیش‌فرض:

```
Essentials
```

استفاده کنید.

---

# درس ۴: فرمت‌های تصویری

## JPG / JPEG

ویژگی‌ها:

✅ حجم کم  
✅ مناسب برای وب  
❌ عدم پشتیبانی از لایه‌ها

کاربرد:

- تصاویر سایت
- شبکه‌های اجتماعی

---

## PNG

ویژگی‌ها:

✅ کیفیت بالا  
✅ پشتیبانی از شفافیت

کاربرد:

- لوگو
- آیکون
- تصاویر بدون پس‌زمینه

---

## PSD

فرمت اصلی فتوشاپ است.

ویژگی‌ها:

✅ حفظ تمام لایه‌ها  
✅ قابلیت ویرایش کامل  
✅ مناسب پروژه‌های حرفه‌ای

---

## TIFF

مناسب برای:

- چاپ حرفه‌ای
- عکاسی
- آرشیو تصاویر با کیفیت بالا

---

# درس ۵: Pixel و Resolution

## Pixel چیست؟

هر تصویر دیجیتال از میلیون‌ها نقطه کوچک به نام Pixel ساخته شده است.

مثال:

```
1920 × 1080 px
```

یعنی:

- ۱۹۲۰ پیکسل عرض
- ۱۰۸۰ پیکسل ارتفاع

---

# Resolution چیست؟

Resolution میزان جزئیات تصویر را مشخص می‌کند.

واحد اندازه‌گیری:

```
PPI (Pixel Per Inch)
```

تنظیمات رایج:

برای نمایشگر:

```
72 PPI
```

برای چاپ:

```
300 PPI
```

---

# تمرین‌های فصل

## تمرین ۱

یک فایل جدید بسازید:

```
Width: 1920 px
Height: 1080 px
Resolution: 72 PPI
Color Mode: RGB
```

---

## تمرین ۲

یک پروژه ساده ایجاد کنید شامل:

- یک پس‌زمینه
- یک متن
- یک شکل

سپس فایل را با فرمت زیر ذخیره کنید:

```
Project-01.psd
```
